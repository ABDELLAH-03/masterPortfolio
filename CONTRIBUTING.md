# 🚀 Contributing to Abdellah Chahdi's Portfolio

Thank you for considering contributing to this portfolio project! Your help is greatly appreciated. 😊

## 🤝 How to Contribute

Here are the steps to get you started:

1. **Fork the Repository** 🍴

   - Click the "Fork" button on the top right of the [portfolio repository](https://github.com/ABDELLAH-03/portfolio) page on GitHub. This will create a copy of the project in your own GitHub account.

2. **Clone Your Fork** 🧙‍♂️
   - In your terminal, run the following command to clone your fork to your local machine:
     git clone https://github.com/ABDELLAH-03/masterPortfolio
3. **Switch to the Project Directory** 📁

   - Move into the project directory using:

4. **Install Dependencies** 📦

   - Install the required packages:
     npm install

5. **Create a Branch** 🌿

   - Create a new branch for your changes with a descriptive name. For example:
     git checkout -b feature/your-feature-name

6. **Make Your Changes** 💡

   - Implement your awesome changes and features.

7. **Test Your Changes** 🧪

   - Make sure everything works correctly before committing:
     npm start

8. **Commit Your Changes** 📝

   - Commit your changes with a descriptive message. For example:
     git commit -m "Added new feature or fixed bug"

9. **Push Your Changes** 🚢

   - Push your changes to your fork on GitHub:
     git push origin feature/your-feature-name

10. **Create a Pull Request** 🙌

    - Go to your fork on GitHub, and click the "New Pull Request" button. Follow the prompts, and submit your changes for review.

11. **Celebrate** 🎉
    - You've contributed to the project! Thanks for your hard work and dedication.
